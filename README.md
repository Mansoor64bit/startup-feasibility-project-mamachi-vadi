# Startup Feasibility Project – Mamachi Vadi

## Project Overview

Mamachi Vadi is a proposed agri-tourism startup located in Goa that aims to provide visitors with immersive farm-based experiences while supporting local farmers and promoting sustainable tourism.

This project was developed as part of the MBA Corporate Finance course and evaluates the financial feasibility of the business using financial modelling and investment appraisal techniques.

---

## Objective

The objective of this project was to determine whether the proposed startup is financially viable and capable of generating attractive returns for investors over a 10-year period.

---

## Tools Used

* Microsoft Excel
* Financial Modelling
* Corporate Finance Concepts
* Capital Budgeting Techniques

---

## Financial Analysis Performed

* Revenue Forecasting
* Operating Cost Estimation
* Cash Flow Projections
* Weighted Average Cost of Capital (WACC)
* Net Present Value (NPV)
* Internal Rate of Return (IRR)
* Profitability Index (PI)
* Payback Period Analysis

---

## Key Assumptions

* Operating Period: 240 days annually
* Revenue generated through day packages and overnight farm stays
* 75% Debt Financing and 25% Equity Financing
* Project Evaluation Horizon: 10 Years

---

## Key Findings

* Positive Net Present Value (NPV)
* Internal Rate of Return (IRR): 46.25%
* Profitability Index (PI): 4.03
* Payback Period: Approximately 3 Years and 10 Months

The analysis indicates that the project is financially feasible and capable of generating strong long-term returns while supporting sustainable rural tourism.
